
# Fionsun Network Website

This repository contains the design and content blueprint for the Fionsun Network website, including Home, About, and Contact pages.

## Overview
Fionsun Network is an AI-powered platform for creative studios, finance solutions, and social media engagement. This project outlines the layout, copy, and visual inspirations for a modern tech website.

## Wiki Pages
- **Home.md** – Layout and copy for the homepage
- **About.md** – Brand story, mission, vision, values, and team
- **Contact.md** – Contact options, form layout, and footer

## Assets
- Placeholder images, logos, and animations (to be added)
- Graphics for sunburst and network visuals

## How to Use
- Review Wiki pages for content
- Use assets for web design mockups
- Implement layout in Webflow, Framer, or WordPress
